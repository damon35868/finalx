<h3 align="center">
  FinalX
</h3>

<p align="center">
基于Taro、React打造的一款跨端开发的工具及组件库
</p>

### Installation 安装

```shell
# npm
npm install @finalx/cli -g

# Yarn
yarn global add @finalx/cli
```

### Installation 生成项目结构

```shell
finalx init <projectName>
```
